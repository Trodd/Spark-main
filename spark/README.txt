SPARK - Ready to Use Release
================================

Thank you for downloading Spark!

QUICK START:
------------
1. Double-click "Spark.exe" to launch the application
2. The application should run immediately with all features enabled

REQUIREMENTS:
-------------
- Windows 10/11 (64-bit)
- .NET 6 Runtime (Desktop) - will be prompted to install if missing

FEATURES INCLUDED:
------------------
- Full EchoVR integration
- Overlay system with all templates in the "Overlay/build" folder
- Discord Rich Presence
- OBS integration
- Replay recording and management
- Speech recognition
- Text-to-speech
- And many more features!

FIRST TIME SETUP:
-----------------
1. Launch Spark.exe
2. Open Settings to configure your preferences
3. Set up your EchoVR connection
4. Configure overlays if needed
5. Start using Spark!

OVERLAYS:
---------
All overlay templates are located in the "Overlay/build" folder.
Open them in your browser or OBS to use them.
